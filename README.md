## Descripción del proyecto

- Este proyecto desarrolla el algoritmo principal y la simulación lógica del funcionamiento de un Panel Direccionable del Sistema Contra Incendios, aplicado al análisis de sensores, eventos, manejo de energía y notificaciones.

- El software está implementado en C#, con estructura modular y simulación de lógica de un sistema real de alarma contra incendios.
